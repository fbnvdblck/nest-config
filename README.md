# NestJS configuration

## Description

A more high level layer for NestJS configuration.

## Set up project

```bash
$ pnpm install
$ pnpm approve-builds
```

## Build project

```bash
$ pnpm build
```

## Quality tools

```bash
# Analyze project code quality with ESLint
$ pnpm lint

# Format project code with Prettier
$ pnpm format
```
